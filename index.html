Proyek

Pembuatan Web Kuliner

Proyek "Kuliner Enak Jawa Timur" (KEJT) adalah sebuah aplikasi web yang menghadirkan pengalaman interaktif untuk mengenal lebih dalam ragam kuliner khas Jawa Timur. Di halaman utama, pengguna disambut dengan video latar belakang yang menarik untuk memberi kesan visual tentang kelezatan kuliner daerah ini. Navigasi yang intuitif memungkinkan akses mudah ke halaman-halaman utama seperti beranda, kuliner, tentang kami, dan kontak. Pengguna dapat membuat akun melalui sistem autentikasi untuk dapat memberikan penilaian serta ulasan pada hidangan favorit mereka, sementara fitur pencarian dan filter membantu dalam menemukan makanan sesuai kategori atau preferensi tertentu. Sistem penilaian dan ulasan ini memungkinkan pengguna untuk berbagi pengalaman kuliner dan membantu pengguna lain dalam membuat pilihan. Pengelolaan konten, ulasan pengguna, dan statistik dilakukan melalui dashboard admin. Proyek ini menggunakan teknologi frontend HTML, CSS, dan JavaScript, backend PHP, serta MySQL untuk database.
